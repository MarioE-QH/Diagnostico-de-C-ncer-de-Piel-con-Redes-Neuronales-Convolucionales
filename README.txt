## 🧠 Skin Cancer Detection with CNN and Deep Learning

Este proyecto utiliza técnicas de inteligencia artificial para clasificar imágenes de lesiones dérmicas en siete categorías, usando el dataset HAM10000. Se implementaron tres modelos CNN personalizado, ResNet50 y EfficientNetB0.

### 📌 Características
- Entrenamiento y evaluación de modelos de clasificación de cáncer de piel
- Métricas comparativas Precisión, Sensibilidad, Especificidad, F1, MCC
- Prueba de McNemar para evaluación estadística
- Matrices de confusión y visualización con mapas de calor
- Generación de reportes personalizados en PDF
- Interfaz web con Streamlit

### 📦 Dataset
- [HAM10000 - Harvard](httpswww.kaggle.comdatasetskmaderskin-cancer-mnist-ham10000)

### 🛠️ Tecnologías usadas
- Python, TensorFlow, Keras, Streamlit, Pyngrok, PIL, FPDF, Matplotlib, Scikit-learn
